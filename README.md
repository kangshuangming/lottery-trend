# 彩票走势分析系统

基于走势分析的彩票选号系统，支持双色球和大乐透，包含数据抓取、多维分析、智能选号、购买记录管理、开奖对比和改进建议功能。

## 功能

- **数据抓取**: 从 500.com 抓取 500 期历史开奖数据（双色球 + 大乐透）
- **多维分析**: 10+ 维度走势分析（频率、遗漏、和值、跨度、奇偶比、大小比、连号、AC值、重号、冷热号）
- **智能选号**: 5 种选号策略（混合策略、热号追势、冷号回补、遗漏回补、均衡组合、智能评分）
- **购买记录**: 手动录入或从选号页面标记已购买，支持取消已购买标记
- **开奖对比**: 基于开奖时间判断，未开奖显示提示，已开奖自动对比并计算中奖金额
- **改进建议**: 号码集中度检测、策略效果分析、冷热号偏离分析

## 技术栈

- Python 3.13 + Flask
- BeautifulSoup4（数据抓取）
- Chart.js（前端图表）
- 原生 HTML/CSS/JS（无框架依赖）

## 安装

```bash
pip install flask beautifulsoup4 requests
```

## 运行

```bash
python app.py
```

打开浏览器访问 http://localhost:5088

## 项目结构

```
lottery-trend/
├── app.py          # Flask 后端 API
├── scraper.py      # 500.com 数据抓取
├── analyzer.py     # 走势分析模块
├── selector.py     # 智能选号模块
├── purchase.py     # 购买记录与开奖对比
├── templates/
│   └── index.html  # 前端页面（6 Tab）
└── cache/           # 运行时自动生成的缓存目录
```
